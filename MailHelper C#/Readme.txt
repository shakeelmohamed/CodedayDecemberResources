This C# class helps you send emails using Gmail SMTP servers.

The "from" field must be a valid Gmail email account, and you must specify your Gmail credentials when calling the mail helper. I recommend you put your email credentials inside your Web.config, resource .resx, or appconfig.config files (or whatever hidden settings options you have).
